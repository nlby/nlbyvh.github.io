学习总结

本学期大部分的时间都贡献给几个课程作业了，也就是进行了一些简单程序的实现，可以说是走了几个开发的流程，但对代码能力的提高是很有限的，因为重复的工作做的有些多。
完成的几个程序

  1. pyqt开发的gui
    樱花动漫的客户端（结合爬虫）
    安全帽检测客户端
    肺炎依赖影像检测客户端
    https://github.com/nlby/pyqt_nlby
  2. app
    学习安卓过程中编写的多个小demo
    cshelper 信息技术学习辅助工具
    https://github.com/nlby/cshelper_nlby
  3. 后端
    cshelper 数据存储系统 （ssm）
    证书信息管理后台 （ssm + spring security）
    bbs (springboot)

学到的

  1. 安卓基本开发
  2. 安卓网络通信
  3. 安卓MVP模式
  4. 文档编写方法
  5. visio基本绘图
  6. QDockWidget与其他控件的结合使用
  7. pyqt多线程实现
  8. python操作数据库
  9. ssm简单项目编写
  10. springboot基本配置与使用
  11. 以太坊原理和基本开发
  12. Fabric原理

需要学的

  1. 提升java编程能力（算法和底层）
  2. pyqt控件的更灵活使用
  3. python web开发
  4. python 爬虫高级使用和数据分析 可视化
  5. 深度学习理论与实践 模型训练测试
  6. 练习 ssm springboot项目 体会设计方法
  7. 从前端基础学起，拥有自制界面的能力 利用网页api的能力
  8. 了解并使用Fabric java sdk
