七月上旬综合学习情况分析与安排

  • 总结
  view

搁了好几天，实际这篇是7.14写的。先总结下过去半个月的工作，然后对之后的工作进行梳理。

半月工作总结

  1. 7.1-7.5 零碎复习一天多 + go io并发 + fabric基础过一遍 + 项目相关权限还有初步整合
  2. 7.6 退出申请
  3. 7.7-7.10 整理java项目 权限 组件 特性等相关的资源，并看了一个后台的介绍教程并进行总结
  4. 7.11-7.12 软件杯yolo资料查找与文档编写 go项目依赖问题解决 sql练习
  5. 7.13-7.14 go iris学习 项目管理测试 + 学习内容整理

整体来说，效率有些过低了。有些迷茫，有些逃避。

后续工作梳理

  1. 首要是复习，八月前，必须彻底完整地好好过一般，这个首要要花大功夫。
  2. 然后是java 新特性 项目进阶 组件 算法这些都要学和练
  3. fabric 要结合 go 想出设计方案
  4. 算法刷题 sql练习

大体来说，最优先的就这些事，但要注意，务必全力以赴，不要让不相干的事互相干扰，切忌徘徊不定，浪费时间。
